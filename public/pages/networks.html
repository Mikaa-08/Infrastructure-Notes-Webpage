What Is a Network?

  • A system connecting devices so they can communicate and share resources
  • LAN: small area (home/school)
  • WAN: large area; the Internet is the biggest WAN

Network Devices

  • NIC: connects a device to the network
  • Hub: broadcasts to all devices (old)
  • Switch: sends data only to the target device
  • Router: connects networks and provides internet access

Network Models

  • Peer-to-Peer: devices share directly, simple and cheap
  • Client/Server: central server stores data and controls access

Protocols

  • HTTP/HTTPS: Web, where the S stands for secure
  • FTP: File transfer
  • SSH: Secure remote access
  • IP: Addressing
  • Bluetooth: Wireless communication

TCP/IP Model

  • TCP ensures reliable, ordered delivery
  • IP handles addressing and routing
  • Data is broken into packets

Testing Commands

  • ping: Test reachability
  • tracert: Shows route
  • ipconfig: View network settings
