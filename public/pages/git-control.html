What Is Git?

  • A version control system used to track code changes
  • Helps teams collaborate and manage project history

Local Repository

  • Created using git init
  • Stores commits, branches, configuration

Basic Workflow

  • git add: stage changes
  • git commit: save snapshot
  • git status: view untracked or changed files

Branching

  • git branch name: create branch
  • git switch name: move to branch
  • Merging combines branches

Remote Repositories

  • Stored on GitHub or GitLab
  • git clone URL: download project
  • git push: upload changes
  • git pull: download updates

Git Setup

  • Windows: install Git installer
  • Linux: sudo apt install git
  • Configure name and email
