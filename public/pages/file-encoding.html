File Types

  • Text files: .txt, .md, .html
  • Binary files: images, executables, videos
  • Executable files: .exe, .app, .sh
  • Document files: .pdf, .docx
  • Media files: .png, .mp4, .mp3

File Extensions

  • Tell the OS which program should open a file
  • Do not determine file content, only formatting

Encoding

  • How characters are represented in binary
  • ASCII: 7-bit encoding, 128 characters
  • Unicode: supports almost every language
  • UTF-8: most common Unicode format

Why Encoding Matters

  • Ensures text displays correctly across devices
  • Prevents corrupted symbols

Compression

  • Reduces file size
  • Lossless: PNG, ZIP (no data lost)
  • Lossy: JPEG, MP3 (some data removed)

Archiving

  • ZIP and RAR combine many files into one
  • Makes transfer easier, supports compression
