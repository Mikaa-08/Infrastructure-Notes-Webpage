Summary

  • Invented by George Boole in the 1800s
  • Based on algebra where values are only true (1) or false (0)
  • Foundation of:
    • Digital circuits
    • Computer programming
    • Decision-making in AI and search engines

Logic Gates

  • AND Gate: Only outputs 1 if both inputs are 1
  • OR Gates: Outputs 1 if at least one input is 1
  • NOT Gates: Inverts the input
  • XOR Gates: Outputs 1 only if inputs differ
  • NAND Gates: Output is the inverse of AND (So output is 0 only if both inputs are 1, otherwise output is 1)

Bits and Data Representation

  • Bit: single binary digit
  • Nibble = 4 bits
  • Byte: 8 bits = 1 character
  • Kilobyte (KB) = 1,000 bytes
  • Megabyte (MB) = 1,000 KB
  • Example: The word "HELLO" (5 characters) = 5 bytes = 40 bits

Transistors and Circuits

  • A transistor acts like a switch -> ON (1) or OFF (0)
  • Billions of transistors combine to form CPUs
  • Circuits of logic gates can math, decision-making, and storing stuff in memory
