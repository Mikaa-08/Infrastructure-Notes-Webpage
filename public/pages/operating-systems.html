What Is an Operating System?

  • Manages hardware, memory, files, and applications
  • Provides a user interface (GUI or CLI)
  • Examples: Windows, macOS, Linux

Memory & I/O Management

  • Uses Memory-Mapped I/O to communicate with hardware
  • Each device is assigned a specific memory address range

File Systems

  • Structures for storing and organizing data
  • NTFS: Windows
  • ext4: Linux
  • APFS: macOS

Processes

  • A process is a running program
  • OS handles scheduling and switching between programs
  • Manages CPU time, memory, and resources

User Interfaces

  • GUI (Graphical User Interface): windows, icons, menus
  • CLI (Command Line Interface): text-based commands

Important Windows Tools

  • Control Panel: system settings
  • Device Manager: drivers and hardware
  • Disk Management: partitions and storage
  • Windows RE: recovery environment

Backups

  • Use File History, cloud backups, disk images

macOS

  • Finder: file manager
  • Dock: fast access to apps
  • Menu Bar: settings and menus
  • Time Machine: backup system

Linux

  • Desktop environments: GNOME, KDE
  • File system: ext4
  • Tools: Terminal, GParted, package managers
