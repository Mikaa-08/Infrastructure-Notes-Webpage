What is a processor

  • CPU (Central Processing Unit): "Brain" of the computer, processes instructions and exectutes them
  • GPU (Graphics Processing Unit): Processes information into graphics to display it
  • Processor speed is measured in GHz, clock speed (Billions of cycles per second), modern CPUs run at 5GHz
  • Main processor manufacturers
    • AMD
    • Intel
    • Nvidia
    • Qualcom

Types of processors

  • CPU:
    • Few powerful cores
    • Good for general-purpose tasks
  • Discrete GPU:
    • Separate card
    • Has its own VRAM
    • Needed for gaming, editing, 3D work
    • Hundreds to thousands of smaller cores
    • Designed for simultaneous math-heavy tasks
  • Integrated GPU (iGPU):
    • Built into CPU, so it shares system RAM
    • Good for low requirement everyday tasks

Core Components of a Core

  • Control Unit (CU)
    • Fetches and decodes instructions
    • Manages communication between ALU, memory, I/O
  • Arithmetic Logic Unit (ALU)
    • Performs arithmetic and logic operations
    • Handles operations like AND, OR, ADD, SUBTRACT
  • Registers
    • Very small, extremely fast storage inside CPU
    • Holds data for immediate use
  • Cache
    • High-speed memory inside CPU
    • Levels:
      • L1 (smallest, fastest)
      • L2
      • L3 (largest, slowest but still very fast)

CPU Cores

A CPU core contains all of the components, technically making each core it's own chip

Numbers of cores on CPUs possible

  • Dual-Core = 2 Cores
  • Quad-Core = 4 Cores
  • Hexa-Core = 6 Cores
  • Octo-Core = 8 Cores
  • 12 Cores
  • 16 Cores
  • 24 Cores

CPU Sockets

  • LGA (Intel)
    • Pins on the motherboard, so CPU has flat contacts
    • Used by most Intel CPUs
  • PGA (AMD)
    • Pins on the CPU, so the socket holes are on motherboard, not with AM5
  • Circuits of logic gates can math, decision-making, and storing stuff in memory

Cooling a CPU

  • Air cooling (fans + heatsink)
    • Cheap and effective for most CPUs
      • Single-tower coolers (1 heatsink + fans)
      • Dual-tower coolers (2 heatsinks + fans)
  • Passive cooling (Just a heatsink, no fans)
  • Liquid Cooling
    • Expensive but more effective than otherLinks
      • Tubing through PC (Expensive + Hard to maintain)
      • AIO (All In One) cooling has fans with a cooling tube

Instruction Cycle

  1. Fetch
    • Instruction loaded from memory to IR
  2. Decode
    • CU interprets instruction
  3. Execute
    • ALU performs operation
  4. Store
    • Result stored in register or memory

Pipelining

  • CPU splits instruction execution into stages
  • Each stage processes a different instruction simultaneously like an assembly line
  • More effective than singhle task computing
