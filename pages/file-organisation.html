Why File Organization is Essential

  • Prevents wasted time searching for files
  • Reduces errors in workplace settings
  • Provides clearness in teamwork: everyone knows where files are stored

Folder Hierarchies

  • Top-level folders represent broad categories
  • Subfolders narrow down into subjects, projects, or dates
  • Good structure helps break down complex projects

Naming Conventions

  • lowercase or camelCase
  • underscores (_) or hyphens (-) instead of spaces
  • Dates in ISO format (YYYY-MM-DD) for sorting

Password Security

  • Weak Passwords = easy to guess (password, birthdays, names)
  • Strong Passwords:
    • Minimum 12 characters
    • Uppercase + lowercase + numbers + symbols
    • Random, different for each account.
  • Best Ways to secure them:
    • Use a password manager (Bitwarden)
    • Enable two-factor authentication (2FA)
    • Change compromised passwords immediately
