Web Browser

  • Software for retrieving and displaying content from the internet
  • Converts HTML, CSS, JavaScript into human-readable pages
  • Examples: Chrome, Firefox, Safari, Edge

Search Engines, How They Work

  1. Crawling: Automated bots ("spiders") scan web pages for keywords
  2. Indexing: The content is categorized and stored in a database
  3. Ranking: Algorithms decide which results are most relevant
    • Factors: keyword matching, site reliability, freshness of content, etc...

Search Operators

  • "word1 word2" = exact phrase
  • word1 AND word2 = must include both
  • word1 OR word2 = either term
  • -word = excludes a term
  • site:example.com = searches within one site
  • filetype:pdf = finds specific file formats
