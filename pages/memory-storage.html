What is Memory(RAM)?

  • Volatile information, means it's lost if power is turned off
  • Used by every program to store temporary information
  • 2 main types of RAM
    1. DRAM (Dynamic RAM):
      • Has to be refreshed thousands of times per second
      • Slower but cheaper, it's used for the main memory in your system
    2. SRAM (Static RAM):
      • Super fast so it doesn't need any refreshing
      • More expensive, so it's used in chache memory inside the CPU
  • Speed of Memory is represented in MHz (Refreshes per second), while capacity is measured in GB

ROM (Read-Only Memory)

  • Is not volatile
  • Stores permanenet instructions for the boot process (Firmware)
  • Motherboards and other devices use ROM for simple instructions
  • Usually uses Flash Memory

Physical RAM types

  1. SIMM:
    • Used in old computers, has one single sided connection (pins)
  2. DIMM:
    • Used in newer computers, has double sided connection

RAM terms

  • SDRAM, sychronises with the system clock to make transfers (Used in DDR RAM)
  • DDR (Double Data Rate), transfers data twice per clock cycle
    • Generations: DDR1; DDR2; DDR3; DDR4; DDR5 (Can't mix generations, only use the one the motherboard supports)
  • PC Number:
    • PC#-XXXXX = DDR# (generation) + XXXXX (max bandwidth MB/s)

CAS Latency

  • Time it takes before the instruction is sent and the data is retrieved
    • Lower CAS latency = faster response time
    • Clock speed also matters, not just CAS latency

Dual-Channel Memory

Use matching sticks in 2 specific slots on the motherboard for it to access both modules simultaneously, doubling the bandwidth

Memory Hierarchy (Fastest-Slowest)

  1. CPU registers
  2. Cache (L1, L2 and L3)
  3. Main system memory (RAM)

Storage:

  • Uses Flash memory for non volatile information
  • Used in BIOS chips and almost all other set device instructions
  • No need for refreshing, means power can go off and the info stays

Storage types

  • SSD (Solid State Drive)
    • Uses transistors and charges to store data
  • HDD (Hard Disk Drive)
    • Uses magnetic polarity to store data and sense changes
  • Optical Drives
    • Uses lasers to sense reflectivness for pits and lands
