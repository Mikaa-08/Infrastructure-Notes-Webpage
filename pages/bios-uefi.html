BIOS (Basic Input/Output System)

  • Firmware stored on the motherboard in flash memory
  • Runs the POST (Power-On Self Test)
  • Initializes basic hardware
  • Loads the bootloader to start the operating system

UEFI (Unified Extensible Firmware Interface)

  • Modern replacement for BIOS
  • Supports mouse and graphical interfaces
  • Supports large hard drives using GPT
  • Features Secure Boot, faster startup, network capability

Boot Process (Startup Sequence)

  1. Power on the computer
  2. CPU loads BIOS/UEFI firmware
  3. POST (Power On Self Test) checks CPU, RAM, GPU, Motherboard
  4. The bootloader is loaded from storage
  5. The operating system kernel loads into memory
  6. Drivers start and the operating systems becomes available

CMOS & Battery

  • Previously stored BIOS settings, now mostly stored in flash
  • Battery (CR2032) keeps the real-time clock running
  • Battery lasts 3-8 years

BIOS Updates (Flashing)

  • Adds support for new CPUs
  • Patches bugs or security issues
  • Improves compatibility with newer hardware
  • Should only be updated when necessary

Boot Problems & Failures

  • Hardware failure (RAM, CPU, PSU, motherboard)
  • Corrupt bootloader or OS files
  • Incorrect BIOS settings
  • Power problems and forced shutdowns

POST Beep Codes

  • 1 short DRAM refresh failure
  • 5 short CPU failure
  • 8 short Display memory error
  • 11 short Cache memory error
  • (Codes vary by manufacturer)
