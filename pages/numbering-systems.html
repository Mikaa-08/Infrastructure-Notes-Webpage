Decimal (Base 10)

  • Digits: 0-9
  • Powers of 10 define place values
  • Example: 526 = (5 x 100) + (2 x 10) + (6 x 1)

Binary (Base 2)

  • Digits: 0, 1
  • Each binary digit = bit
  • Example conversion: Decimal 23 → Binary:
    • 23 ÷ 2 = 11 remainder 1
    • 11 ÷ 2 = 5 remainder 1
    • 5 ÷ 2 = 2 remainder 1
    • 2 ÷ 2 = 1 remainder 0
    • 1 ÷ 2 = 0 remainder 1
    • = Binary = 10111

Octal (Base 8)

  • Digits: 0-7
  • Compact binary form (3 bits per octal digit)
  • Example: Binary 11010110 → (110)(101)(10) → 326

Hexadecimal (Base 16)

  • Digits: 0-9 and A-F
  • Compact binary form (4 bits per hex digit)
  • Example: Binary 11111111 = Hex FF
  • Used in:
    • Memory addresses (0x7FFF)
    • Color codes in HTML (#FFFFFF = white)

Conversion Techniques

  • Decimal → Binary: repeated division by 2
  • Binary → Decimal: multiply digits by 2ⁿ and sum
  • Binary → Octal: divide into groups of 3 and assign the decimal values
  • Binary → Hexadecimal: divide into groups of 4 and assign decimal values
