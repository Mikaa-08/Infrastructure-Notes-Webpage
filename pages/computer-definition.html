What is a Computer?

  • A computer is a programmable electronic device that:
    1. Receives input from a user or program.
    2. Processes data based on stored instructions.
    3. Stores information temporarily or permanently.
    4. Generates output for the user or the program.
  • An important feature is programmability, the same hardware can be reprogrammed to do completely different tasks

Characteristics of a Computer

  • Speed: Modern CPUs can perform billions of calculations per second
  • Accuracy: Errors only occur due to programming mistakes or hardware malfunctions
  • Automation: Once given instructions, a computer executes without human intervention
  • Storage Capacity: Can store vast amounts of data in small spaces
  • Connectivity: Most computers are networked, enabling communication accross long distances

Computer Components

  • Hardware (physical parts):
    • CPU (Centrak Processing Unit)
    • RAM (Random Access Memory)
    • Hard Drive / SSD (storage)
    • Input devices (keyboard, mouse, camera, sensors)
    • Output devices (monitor, speakers, printer)
  • Software (instructions/programs):
    • System Software, operating systems (Windows, macOS, Linux).
    • Application Software, apps and programs (Micosoft Word, Google, NotePad++)
    • Utility Software, antivirus, file managers, backup tools.
